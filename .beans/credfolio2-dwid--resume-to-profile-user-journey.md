Complete user flow from resume upload through profile enhancement and PDF export.

## User Journey

1. **Upload Resume** → See skeleton loading state → View extracted profile
2. **Enhance with Reference Letters** → See visual diff of what was added → Accept/reject changes
3. **Manual Editing** → Fine-tune any field → Undo mistakes
4. **Export** → Generate professional PDF

## Success Criteria

- First-time user can go from upload to viewing their profile in under 60 seconds
- Clear visual feedback during async processing
- Obvious indicators of what reference letters contributed
- Easy to undo any change
- PDF export looks professional and polished
